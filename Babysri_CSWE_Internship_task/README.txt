Project Title: Drone Telemetry Simulation Dashboard
Role: CSWE Intern (Software)

This project is a web-based simulation of a drone telemetry system.
It displays live drone parameters such as altitude, speed, battery
percentage, and GPS coordinates.

The telemetry updates every 1 second for 30 seconds, simulating
real-time drone behavior.

Contents:
- Code folder: HTML, CSS, and JavaScript files
- Documents folder: One-page project explanation

Tools Used:
- HTML, CSS, JavaScript
- GitHub Pages (for hosting)

This project was created as part of the CSWE Internship Screening Task.
